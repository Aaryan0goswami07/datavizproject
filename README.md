# Complete Data Analysis and Visualization Project

## Overview
This project demonstrates a full data analysis pipeline using Python. It includes data loading, cleaning, analysis, and visualization to extract meaningful insights from a real dataset.

## Features
- Data loading and validation
- Data cleaning (missing values & duplicates)
- Statistical analysis of numerical data
- Data visualization using bar and pie charts
- Error handling for file loading

## Tools Used
- Python
- Pandas
- Matplotlib

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

Visualizations will be saved in the **visualizations** folder.
